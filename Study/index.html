# VuePress静态文档类建站

# 1.0 Hosts文件修改

# 文件位置：

C:\Windows\System32\drivers\etc

# 修改原因：

无法解析 xxx.github.io

例如：assistantliu.github.io

# 改动内容：

最后一行加入：ip xxx.github.io

例如：185.199.108.153 assistantliu.github.io

# 命令行：

powershell输入：Clear-DnsClientCache

# 查询ip:

https://www.ipaddress.com/

https://www.ipip.net/ip.html-

# 关闭防火墙

关闭当前防火墙

# 代理服务器

请勿通过代理访问，也会访问不到。

例如：公司电脑无法连接外网，我通过另一台电脑开放CCProxy来进行上网，那么如果我通过连接代理的方式上网来访问github.io是访问不到的，即便是改了hosts文件.（实测）
